## Mission
The umbrella marketing, documentation, and community site for the **Train Better** ecosystem. The front door that routes athletes, coaches, and curious visitors to the right product.

## Core Requirements
1. **Product showcase**: Explain what Logbook Companion and ErgLink do, with clear CTAs
2. **Documentation home**: User guides, getting started, FAQ
3. **Community hub**: Public roadmap, feature requests, feedback forms
4. **Auth landing**: Shared signup/login that routes users to the correct app post-auth
5. **SEO-friendly**: Server-rendered pages (Next.js App Router) for discoverability

## Non-Negotiable
- Must not duplicate workout/coaching logic from LC
- Must share Supabase Auth with LC and EL (single auth project)
- Must deploy to Vercel on `readyall.org` domain
- Must be responsive (mobile-first)

## Target Users
- **Athletes** looking for a workout logging solution
- **Coaches** evaluating team management tools
- **Existing users** needing docs, support, or feedback channels
- **Search traffic** looking for rowing/erg training tools
